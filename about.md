About this software
===================

pdtk 0.0.0
----------------

### Authors

- R. S. Doiel

pdtk is a Pandoc preproccessor. The primary use case is rendering JSON
documents via Pandoc templates.

### Programming languages

- Go

### Operating Systems

- Linux
- Windows
- macOS

### Software Requiremets

- Pandoc 2.18 or better
- GNU Make (recommended)
