NAME

pttk gs

SYNOPSIS

pttk gs [HTDOC_PATH] [OPTIONS]

DESCRIPTION

pttk gs provides a simple static gopher server for testing the content you’re Gopher content.

EXAMPLE

In the example the htdoc directory is called “myblog” and you can view the result at gopher://localhost:7000.

pttk gs $HOME/Sites/myblog
